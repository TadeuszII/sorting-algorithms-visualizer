# Sorting Algorithms Visualizer

Python desktop application for visualizing sorting algorithms with animations, interactive controls, CSV import/export, and live performance metrics.

## Features

- Visualization of 14 sorting algorithms
- Step-by-step and automatic execution
- Start, stop/resume, reset, and delete controls
- Real-time metrics: comparisons, swaps/writes, execution time
- Data generation in multiple modes
- CSV import and export
- Clear bar-based graphical interface

## Algorithms

- Bubble Sort
- Bubble Sort Flag
- Insertion Sort
- Selection Sort
- Quick Sort
- Quick Sort (Median of Three)
- Merge Sort
- Heap Sort
- Shell Sort
- Intro Sort
- Cocktail Shaker Sort
- Odd Even Sort
- Gnome Sort
- Bitonic Sort

## How to run

To run the project, compile `main.py`:

```bash
python main.py
```

## Notes

- Bitonic Sort requires the number of elements to be a power of 2.
- Recommended resolution: 1920x1080.
